# ⚠️ IMPORTANT: Run From File Explorer

If you are viewing this folder in SharePoint, OneDrive, or your web browser:

- **Do NOT run any setup or app files directly from the browser or SharePoint.**
- **Download the entire folder to your computer.**
- **Open the folder in Windows File Explorer.**
- **Double-click the setup and app files from File Explorer only.**

This ensures proper installation and launching of all forecasting applications.

For details, see SETUP_GUIDE.html.
